# punchChartJS

## What is it?

punchCardJS is a simple js graph that displays the number of actions in a certain hour of a certain day.

## Default settings

data object:

  ```object = {
      1: array[23],
      2: array[23],
      3: array[23],
      4: array[23],
      5: array[23],
      6: array[23],
      7: array[23],
     };
     ```
     
 where object name - day of week, the array contains the number of actions in opredlenie hour.
